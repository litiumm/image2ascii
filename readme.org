* /Compressing the image/

How to compress ?
    Suppose a 400*400 image, group 2x2 pixels and find their average value.
    So iterate through each (x,y) pixel, find the average of that (x,y) pixel, (x+1,y) pixel, (x+1, y+1) pixel and (x,y+1) pixel.

** Pseudocode

Array Image -> elements Pixel
Pixel structure {
        int r, b, g;
}

arr image <- image as an array
arr new_image <- compressed image as an array
